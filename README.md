Robovie-i Ver.2 & V-duino-i
=========================
このプロジェクトは入門用ロボットキット「Robovie-i Ver.2」の3Dプリント版の造形データと、  
ワイヤレスロボット制御ボード「V-duino」を搭載した二足歩行ロボット「V-duino-i」のサンプルプログラムです。

ディレクトリ一覧
----------------------
本プロジェクトで用意されているディレクトリ一覧は下記の通りです。
1. Robovie-i Ver.2_3D_print_stl  
3Dプリント版Robovie-i Ver.2のSTLデータ
1. V-duino-i_HTML_Controller  
V-duino-iをスマートフォンやPCのブラウザから操作するためのサンプルプログラム

環境構築方法や、サンプルプログラムの解説は製品のダウンロードページより、以下のファイルを参照してください。

* V-duino チュートリアル.pdf
* V-duino 取扱説明書.pdf
